<h1>
  Attendance Tracker for KMICS UG
</h1>
<body>
  This website is an attendance tracker and analyzer for the students of KMICS UG.
  The users can create an account and mark their daily attendance manually and the website analyzes their attendance percentage and also gives suggestions to hit their attendance targets.

  <p> 
  We are using the below stack mentioned to complete this project. <br> 
  Frontend: <br> 
  HTML <br>
  CSS <br>
  Javascript <br>
  React.JS <br>
  Next.JS <br>
  Backend: <br>
  Supabase <br> 
  PostgreSQL <br>
